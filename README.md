## Sobre
O projeto reúne diversas rotinas escritas em Javascript para conversão, validação e formatação de textos, números e datas.

## Rotinas
As rotinas disponíveis no projeto são:

* Conversão
    * Segundos para horas
    * Horas para segundos
* Data
    * Extrair dia
    * Extrair mês
    * Extrair ano
* Extenso
    * Extenso de valor monetário
    * Extenso data
* Formatação
    * CPF e CNPJ
    * Nome próprio
* Validação
    * CPF e CNPJ
    * Data
    * Email
    * Ano bissexto
    * Nome próprio
